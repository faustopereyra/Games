# Flappy Bird
Replay the classic Game with this clone! Build with the p5.js library it's a replica of the original game.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install Flappy Bird.

```bash
npm install 
```

## Usage

```bash
npm run live-server
```
